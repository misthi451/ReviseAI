## 🐛 Troubleshooting Quick Fixes

| Problem | Solution |
|---------|----------|
| Frontend not loading | Open in modern browser (Chrome, Firefox, Safari) |
| "Cannot connect to API" | Make sure backend is running: `npm start` |
| "Invalid API key" | Check .env file - key must start with `sk-ant-` |
| "Port 5000 in use" | Change PORT in .env or kill process |
| "CORS error" | CORS already enabled - check endpoint URL |
| "No results" | Check internet, try simpler input, verify API key |
